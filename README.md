# ControlAcademico

This repository contains the Android Gradle project inside `AndroidApp/`.

If you run Gradle commands from the repository root, point Gradle to that folder:

```bash
./AndroidApp/gradlew -p ./AndroidApp <task>
```

Examples:

```bash
./AndroidApp/gradlew -p ./AndroidApp help
./AndroidApp/gradlew -p ./AndroidApp assembleDebug
```

You can also `cd AndroidApp` and run `./gradlew <task>` directly.
